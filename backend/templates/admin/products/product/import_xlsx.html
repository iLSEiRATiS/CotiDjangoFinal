Importar productos vía XLSX

Esta pantalla trabaja con el formato real de la planilla de productos de Coti. La descarga de productos existentes respeta la misma estructura del archivo base y agrega al final los productos nuevos que todavía no estén en esa lista.

Recomendación: primero descargá el archivo actual, editá esa misma planilla y luego volvé a importarla.

Archivo de referencia

La importación y la exportación están alineadas con el archivo Exportacion-productos-03-02-26.fixed.xlsx.

Formato requerido

La fila 1 debe mantener estos encabezados y este mismo orden:

{{ export_headers|join:", " }}

El archivo esperado tiene 20 columnas, con categorías en una sola columna como Categoria > Subcategoria > Seccion y las imágenes en URL IMAGENES.

Descarga recomendada

Descargar productos existentes

Usá esta descarga como base para editar precios, stock, nombres, categorías, atributos o imágenes sin romper el formato.

Cómo procesa la importación

  • Se toman como referencia principal campos como Nombre, Precio, Categorías, Stock, Mostrar en tienda y URL IMAGENES.
  • Los atributos se leen desde Nombre atributo 1/2/3 y Valor atributo 1/2/3 cuando existan.
  • Si una categoría no existe, se crea automáticamente siguiendo la ruta indicada en la columna Categorías.
  • Nuevos: productos creados. Actualizados: productos existentes modificados.
  • Para evitar errores de formato, no conviene subir planillas vacías ni archivos armados desde cero.
